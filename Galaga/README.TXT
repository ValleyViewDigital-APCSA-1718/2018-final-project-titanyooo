PROJECT TITLE: Galaga
PURPOSE OF PROJECT: Space shoooter
VERSION or DATE:5/30/18
HOW TO START THIS PROJECT: Open program and click start/run
AUTHORS:Titan Lam
DESCRIPTION: Galaga is a classic aarcade game where the player is in control of a spaccraft which is at the bottom of the screen. The player will encounter aliens/enemies trying to destroy the player. Each time the player destroys all aliens/enemies on the screen, the game progresses and gets more difficult. The player has 3 lives and once the player loses all their lives, it's game over. The objective of the game is to get a high score along with passing all the levels. 
HOW TO PLAY: Players will use the left and right arrow keys in order to move horizontally across the levels. The spacebar is used to shoot.
SPECIAL FEAUTURES: Each alien gives a different amount of points. One of the aliens has two phases in which the player must hit two times in order to destroy it and gain the points.
